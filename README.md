# Fuel App Frontend

## Project Description
Fuel App Frontend is a web application designed to help users easily manage their fuel consumption and expenditures. The application provides insights, analytics, and management tools for tracking fuel usage.

## Features
- User authentication
- Fuel consumption tracking
- Expense tracking
- Visual analytics dashboard
- Responsive design

## Tech Stack
- React
- Redux
- Node.js
- Express.js
- MongoDB

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/espeescobar/fuel-app-frontend.git
   ```
2. Navigate into the project directory:
   ```bash
   cd fuel-app-frontend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage Guide
- After starting the development server, open your browser and go to `http://localhost:3000` to view the application.
- Register for an account or log in to access the features.
